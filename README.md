# Password Strength Checker

A simple Python program that checks how strong a password is.

## Features

- Checks password length
- Detects numbers
- Detects uppercase letters
- Detects special characters
- Classifies password as Weak, Medium, or Strong

## How to use

1. Run the program
2. Enter your password
3. See the result

## Example

Input:
mypassword123

Output:
Medium password
